The system includes a custom tmux configuration (`.tmux.conf`) optimized for agent orchestration and Claude Code sessions.

### Navigation
- **Alt + Arrow Keys**: Move between panes
- **Ctrl + Arrow Keys**: Switch between windows
- **Alt + 0-9**: Jump to specific window by number
- **Shift + Arrow Keys**: Resize panes

### Window Management
- **Ctrl-a + |**: Split pane horizontally
- **Ctrl-a + -**: Split pane vertically
- **Ctrl-a + n**: Create new window
- **Ctrl-a + x**: Kill current pane
- **Ctrl-a + X**: Kill current window

### Session Management
- **Ctrl-a + d**: Detach from session
- **Ctrl-a + r**: Reload config
- **Ctrl-a + C-x**: Kill entire session
