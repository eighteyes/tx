## NOTES FOR AGENT

When tailoring this resume:

1. **Reorder sections** based on job requirements (e.g., move Projects before Experience for project-heavy roles)
2. **Select relevant bullets** from each job - don't include everything
3. **Emphasize matching skills** from the Technical Skills section
4. **Use similar language** to the job description where appropriate
5. **Adjust summary** to mirror key requirements from the posting
6. **Include metrics** where possible (%, $, time saved, users impacted)
7. **Keep it focused** - 1 page for <5 years experience, 2 pages for more
8. **Match tone** - more formal for enterprise, more casual for startups

Remember: Quality over quantity. Every bullet should add value for THIS specific role.
